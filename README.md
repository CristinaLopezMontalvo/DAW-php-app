# Desplegament-web
Aquesta és una petita aplicació web feta amb PHP.

## Comandos básicos de Git utilizados

- 'git init' → Crea un repositorio vacío en tu carpeta.
- 'git clone <url>' → Copia un repositorio de GitHub a tu ordenador.
- 'git status' → Muestra el estado de los archivos (qué ha cambiado).
- 'git add <archivo>' → Prepara un archivo para el commit.
- 'git add .' → Prepara todos los archivos modificados.
- 'git commit -m "mensaje"' → Guarda los cambios con un mensaje.
- 'git branch' → Muestra o crea ramas.
- 'git checkout <rama>' → Cambia de rama.
- 'git pull' → Descarga y actualiza con los últimos cambios del remoto.
- 'git push' → Sube tus commits al repositorio remoto.
